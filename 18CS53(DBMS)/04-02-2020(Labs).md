## SQL Constraints

### Col level constraints
* CREATE TABLLE table_name(
    col1 datatype constriants,
    col2 datatype constraints
)

* NOT NULL 
* UNIQUE
* PRIMARY KEY:- combination of not null and unique
* FOREIGN KEY:- uniquely identifies a  row/record in another tabl
* CHECK:- Ensures all values in a column satisfies a specific condition
* DEFAULT:- Sets a default value for a column when no value is specified
