# Dictionaries

- mutable
- key value pair
- not ordered
- cant be sliced
- key error
- keys() list of keys, values() list of values, items() tuple of key and value
- in and not int with dict.keys() , dict.values()
- get() to get values of keys .get(key, fallback)
- setdefault() key, value if key not exist setdefault(key, value)

```
message = 'hi saaketh'
count = {}
for characheter in message:
    count.setdefault(character, 0)
    count[character] = count[character] + 1
print(count)
```

- update({4: 'four'}) adds key value to dictionary

```
def concat(*args):
    dic3 = {}

    for dic in args:
        for x, y in dic:
            dic3.update(x, y)
    return dic3

```
