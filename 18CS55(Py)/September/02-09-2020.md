## Python
* Loosely typed
* More built in fncs
* No braces to define blocks
* Simpler syntax
* : is used when elements spans over one line
* indented

* 2 + 2 => expression
* 2 => expression

## operators
* ** -> exponent
* // -> integer division
* Precedence (Order of operation) ** /*+-
* Evaluated from left to right
* Use paranthesis to override precedence

## Data types
* Integers :- -2, -1, 0, 1, 2, 3
* Floating-point numbers:- 1.25, -1.25
* String:- 'Hello', "Techie", `Saaketh`

## String concatenation and replication
* when + is used two string values, it joins the strings. -> String concatenation
* \* operator is used for string replication. an integer with string replicates string.
