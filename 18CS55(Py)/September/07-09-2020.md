# Bllocks of code

- Begins with indentation
- can contain other blocks(nested blocks)
- ends when indentation ends to zero or to containing block's indentation

# Flow control stmts

## if stmts

- if keyword
- a cond
- a colon
- and indented block of code

## else stmts

- else keyword
- colon
- clause

## elif stmts

- AN elif stmt always consists of the following
  - the elif keyword
  - condition
  - a colon
  - starting on the next line, an indented block of code(called the elif clause)

## while loop stmt

- a while stmt always consists of the following:
  - while keyword
  - cond
  - colon
  - clause

# break stmst

- to get out of the clause

# continue stmt

- to go to the start of the loop

# For loops and the range() fn

- looks like s in range(5): has following
  - the for keyword
  - var name
  - in keyword
  - range
  - colon
  - clause

# range fn

- one param -> 0 to param
- two params -> 1st param till 2nd param excluding 2nd param
- three params -> 1st param till 2nd param excluding 2nd param in steps of 3rd param
