# Handling Absolute and Relative Paths

- os.path.abspath(path) will returns a string of absolute path of the argument.
- os.path.isabs(path) returns abs ? true : false
- os.path.relpat(path, start) => returns relpath based on start if start is not provided start = cwd
- os.path.dirname(path) -> returns stirng of everything that comes before the last slash in the path arg.
- os.path.basename(path) -> returns string of everything that comes after the last slash in the path argument.
- os.path.split(fiilepath) ->returns tuple of basename and dirname
