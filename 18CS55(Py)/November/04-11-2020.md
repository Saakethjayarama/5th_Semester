# the file reading/writing process

- .txt and .py extention files are only plaintextfiles
- 3steps
  - a call to open fn to return file object
  - read or write
  - close the file by calling the close method on the file object.
- r -> read
- w -> w
- a -> a
- .read() reads entire content
- .readlines() -> returns list of string values each string is a line appended with \\n
