# File permissions

10 characters

- In unix we got 3 types of permissions for files

1. Owner or user permission
2. Group permission
3. Others permission

## Symbolic

- rwx - read, write and execute
- default file permission rw-rw-r--
- Symbolic mode(u-user, g-group, o-others)
  (r-read, w-write, x-execute)

## Absolute mode

0 - 000 - no permission
1 - 001 - only execute
2 - 010 - only write
3 - 011 - only write and execute
4 - 100 - only read
5 - 101 - only read and execute
6 - 110 - only read and write
7 - 111 - all read, write and execute
