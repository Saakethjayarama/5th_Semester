# pattern matching functions in UNIX.

- GREP:- Global/regular expression/printing
- The grep filter searches a file for a particular pattern of characters and displays all lines that contains that pattern.
- The pattern that is searched in the files is referred as the regular expression.

- syntax:-
  ```
      grep [options] pattern [files]
      grep -c 'unix' test.txt
  ```

# Options

- -c :- prints only the count of the lines that matches the pattern
- -h :- displays the matched lines, does not display the file name+
- -i :- Ignores the case
