# Responsibilities of shell

1. Program execution
2. Variable and file substitution
3. IO Redirection
4. Pipeline hookup
5. Environment control
6. Interpreted programming language
   - assignment statement
   - conditional statement(if, switch)
   - repeatitive statements(For, while, do while)
   - break, continue, return

# shell variable

1. System defined variables(\$PATH is an ex of sys defined var)
2. User defined variables(x=25, here \$x is user defined variable)
3. \$PATH :- whenever a command is typed on the propmt and the enter command is pressed the shell searches for the command in the shell for internal commmands next it the searches the folders given in the path variable
