# touch

```
touch filename
? -> matches a single char
* -> matches one or more chars
[] -> matches any character in the list
- -> from and to
```

```
ls ? //lists file with only one character
ls ?? // lists file with only two characters
rm ? // removes files with one character

ls *a //prints files ending with a

ls [abcd]* //lists all the files which starts wit either of chars from list

ls [a-c]* lilsts all the files starting from any of a to c
```
